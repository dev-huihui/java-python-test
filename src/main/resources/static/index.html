Java ↔ Python 검증 데모

검증 규칙은 Python 파일에만 있고, 실행 위치(프론트/백엔드)만 다릅니다.

회원가입 검증 (validation.py)

① 프론트엔드 검증 Pyodide (브라우저)

브라우저가 Python 을 직접 실행합니다. 서버에 Python 설치가 필요 없습니다.

② 백엔드 검증 ProcessBuilder (서버)

서버가 python cli.py 를 호출해 검증합니다. 서버에 Python 설치가 필요합니다.

다양한 검증기 (validators.py)

③ 여러 검증 기법 프론트 / 백엔드 선택

정규식·체크섬(Luhn/사업자번호)·표준 라이브러리(IP/날짜/URL)·점수 기반 등 다양한 검증을 실행 위치를 골라 확인합니다.
